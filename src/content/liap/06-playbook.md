# 🎯 Outreach Playbook

← [[00 - Home]]

---

## The Full Funnel

```
Week 1
  → Run Connection Automation (target companies)
  → 10–20 requests/day, SAFE mode

Week 1–2 (as people accept)
  → Run Msg Prospect
  → Message new connections within 24–48h of connecting
  → Cap: 5–10/run

Week 2–3
  → Run Follow Up (wait_days = 3)
  → One gentle nudge to silent connections
  → Cap: 5/run
```

---

## Connection Request Tips

**What works:**
- Target specific roles: CPO, Head of Product, VP Engineering, Founder
- Keep company list focused (5–10 companies per run)
- Use SAFE mode — LinkedIn flags bursts of activity

**What to avoid:**
- Sending to 2nd/3rd connections you have nothing in common with
- Running multiple times in the same hour
- Connecting with more than 20 people/day

---

## First Message Tips

**Timing:**
- Best to message within 24–48 hours of connection
- Avoid Mondays (high inbox volume) and Fridays (end of week)

**Message quality checklist:**
- [ ] Opens with their first name
- [ ] References their actual role
- [ ] Genuinely curious, not transactional
- [ ] No mention of jobs, recruiting, or pitching
- [ ] Under 3 sentences
- [ ] Ends with a soft question or open statement

**Example of a good first message:**
> Hi Tomer, nice to connect! I'm curious about the strategy challenges that come with scaling a product org — what's been the most interesting problem you've worked through recently?

---

## Follow-Up Tips

**Timing:**
- 3–5 days after first message is the sweet spot
- If no reply after follow-up, stop. Don't send a third message.

**Follow-up message quality:**
- Even shorter than the first message (1–2 sentences)
- Acknowledge they're busy — don't guilt trip
- Give them an easy out ("no worries if timing isn't right")

**Example of a good follow-up:**
> Hi Tomer, just wanted to bump this in case it got buried — no worries at all if now isn't a good time!

## LinkedIn Safety Rules

> [!danger] Protect Your Account
> LinkedIn permanently bans accounts that trigger their automation detection. These rules keep you safe.

| Rule | Limit |
|------|-------|
| Connection requests per day | ≤ 20 |
| Connection requests per week | ≤ 100 |
| Messages per day | ≤ 20 |
| Always use | SAFE or NORMAL mode |
| Never | Run overnight / unattended for hours |
| Never | Use DEMO mode for real outreach |

---

## Interpreting Results

### What "Replied ✓" means
They sent a message back. This is a warm lead — follow up manually, take the conversation off automation.

### What "Followed Up" means
You sent the automated follow-up. Ball is in their court. Don't contact again automatically.

### What "Pending" means
First message sent, waiting for reply or follow-up window to expire.

## Recommended Weekly Rhythm

| Day | Action |
|-----|--------|
| Monday | Run Connection Automation (10–15 targets) |
| Tuesday | Run Msg Prospect (new accepts from Monday) |
| Wednesday | Run Msg Prospect + check Follow Up tab |
| Thursday | Run Follow Up if 3+ days since first messages |
| Friday | Review replies, respond manually |
| Weekend | Nothing — let LinkedIn rest |
